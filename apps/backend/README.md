## Installation

```bash
$ npm install
```

## Add OPENAI API key
1. Go to OpenAI website, create and obtain a new secret API key.
2. Add this API key in .env in the following format:
```
OPENAI_API_KEY={the key}
```

## Running the app 

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
